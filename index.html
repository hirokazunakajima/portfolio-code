ABOUT

Cording is interesting. Thinking of cording idea always makes me curious. On the other hand, I like UI / UX design too. I respect designer’s passionate work and when I collaborate with them, I'll try to understand their mind or strategy to make good work happen.

  • ANALYSIS

    Breaking the project into pieces especially for coding part to figure out critical point which might be obstacle while I’m tackling. It takes time but it's worth to understand.

  • ORGANIZE

    Thinking about time schedule based on analysis to get my work on schedule. I’m going to do this one carefully because I need to blow up my coding imagination more.

  • DEVELOPMENT

    Dive into cording to make it real. Actual cording starts from this phase. I take care of my code as much as possible at the same time pay attention to efficient cord as well.
